# Hi, I'm Tri 👋

I'm a young game builder learning how to turn ideas into playable games with code and AI.

Right now, I'm exploring **JavaScript**, **HTML5 Canvas**, and **AI-assisted coding** to build small games inspired by the games I love.

## 🎮 What I'm building

My first project is a simple **Geometry Dash-inspired mini game**:

* A square player that runs automatically
* Jumping with the keyboard
* Spikes and obstacles
* Game over and restart
* Simple levels designed by me

## 🌱 What I'm learning

* JavaScript basics
* Game loops
* Gravity and jumping
* Collision detection
* Level design
* Debugging step by step
* How to use AI as a coding assistant, not as a replacement for thinking

## 🧠 My learning rule

> AI can help me write code faster, but I still need to understand what the code does.

## 🚀 My goal

To build small, fun games, improve a little every week, and share what I learn along the way.
